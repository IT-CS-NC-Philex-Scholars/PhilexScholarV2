PhilexScholarV2 is designed to serve two main user groups: **Students** and **Administrators**.

-   **Application Wizard**: Step-by-step guidance from draft to submission.

## üîÑ The Scholarship Lifecycle

Every application follows this strict path:

1.  **Draft**: Student starts application.
2.  **Submitted**: Sent for review.
3.  **Documents Review**: Admin checks files.
4.  **Eligibility Check**: Admin checks grades/units.
5.  **Enrolled**: Student is accepted.
6.  **Service**: Student performs community service.
7.  **Disbursement**: Student receives funds.
