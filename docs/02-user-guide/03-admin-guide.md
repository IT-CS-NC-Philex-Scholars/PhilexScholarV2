# Admin User Guide

This guide details how to manage the scholarship system, review applications, and process payments.

## 1. Dashboard Overview

The **Admin Dashboard** gives you a bird's-eye view:
-   **Statistics**: Total Students, Active Applications, Pending Reviews.
-   **Recent Activity**: A list of the latest submissions needing attention.
-   **Quick Actions**: Shortcuts to create programs or review docs.

## 2. Managing Scholarship Programs

Navigate to **Scholarships**.

### Creating a Program
1.  Click **"Create New Scholarship"**.
2.  **Budgeting**:
    -   Enter `Total Budget` (e.g., 1,000,000).
    -   Enter `Per Student Budget` (e.g., 10,000).
    -   *System Automation*: The system automatically calculates `Available Slots = 100`.
3.  **Eligibility**:
    -   **School Type**: High School Only, College Only, or Both.
    -   **Academic**: Minimum GPA, Minimum Units.
    -   **Service**: Required Community Service Days (Default: 6).
4.  **Requirements**: Add document types students must upload (e.g., "Barangay Indigency", "Grade 12 Report Card").

## 3. Reviewing Applications

Navigate to **Applications**. You can filter by status (e.g., "Pending Review").

### Phase 1: Document Review
1.  Open an application with status **Documents Under Review**.
2.  You will see the uploaded files on the screen.
3.  **Action**:
    -   **Approve**: File is valid.
    -   **Reject**: File is blurry or wrong. *Select a rejection reason* (e.g., "Unreadable"). The student will be notified to re-upload.

### Phase 2: Eligibility & Enrollment
Once documents are approved:
1.  The system status moves to **Eligibility Verified** (if auto-check passes) or requires manual check.
2.  Review their GPA/Units against the program rules.
3.  Click **Enroll Student** to accept them into the program.
    -   *Note*: This deducts 1 slot from the program's available slots.

## 4. Community Service Review

Navigate to **Community Service**.
1.  Review submitted logs or PDF reports.
2.  Check the photos/evidence provided.
3.  **Approve**: Hours are added to the student's progress.
4.  **Reject**: Provide a reason.

## 5. Disbursements

When a student completes service, they appear in the **Disbursements** tab.
1.  Review the student's completion summary.
2.  Process the payment (e.g., via Cheque or Bank Transfer).
3.  Enter the **Reference Number** (e.g., Cheque No.).
4.  Mark as **Disbursed**.
